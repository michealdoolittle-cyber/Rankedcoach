Current Theme Builder guidance
Use this order when making edits yourself:
1. Start with `Parent Cards`
- Use this for overall card background, child card color, child text, font, overlay sizing, and whole-card content shift.

2. Then use `Containers & Inner Text`
- Use this for the visible container block and the text inside it.
- This is now the preferred everyday workflow for:
  - shells
  - grouped content blocks
  - readable text adjustments
  - safe spacing/size refinements

3. Only use `Show Advanced Elements` when absolutely necessary
- This mode is heavier and is meant for rare one-off nested target fixes.
- It is not the recommended path for routine maintenance edits.
